# Collections

| Type | Notes |
| --- | --- |
| `Vector` / `List` | Contiguous; indexing O(1) |
| `HashMap` / `HashSet` | Expected O(1) lookup/insert/remove |
| `Queue` / `Stack` / `RingBuffer` / `BitSet` | Explicit capacity/bounds behavior |

See `docs/Vela-collections.md` for the detailed collection guide.
